BOARDGAME Project

Full Stack

Context

During this project session, we set out to develop an architecture that would make it easy to develop any boardgame.
So we built a tic-tac-toe as a MERN web application. It runs on React on the front end, and on Node and Express on the back end. Furthermore, we're using MongoDB, a NoSQL database, to make the API fully dynamic and the data fully persistent.

Source code & Docker Image of the application

The source code is accessible via the GitHub directory, which you can find here.
You can also download the application image in the Docker container here.

How to run the game?

Start by creating and then running the containers with the "docker-compose up" command.
Next, open two browsers and connect to your port 3000 ('localhost:3000').
Now you're ready to play.
