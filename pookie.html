Poker Assistant

Neuron Network

Context

This project aims to create an assistant to play poker in online tournaments.
Pookie by name, has to guide the player on the reaction he should have to maximise his chances to win
This way, the AI must suggest to fold, check, call, bet or raise depending on the situation, and indicate a chip amount in case of a bet or a rise.

Code source of the poker assistant

The source code is accessible via the GitHub directory, which you can find here.
You can also download the application folder here.

How to run the Poker assistant?

Start by downloading the zip file above.
Next, run the JupyterNotebook createDataBase to create the dataset that will be used to train Pookie.
Run the JupyterNotebook createModel to train Pookie.
Now you can inform about the features Pookie is asking you for.
