Projet BOARDGAME

Full Stack

Contexte du projet

Nous avons entrepris, au cours de cette session de projet de mettre au point une architecture à partir de laquelle il est aisé de développer n’importe quel boardgame.
Nous avons donc construit un morpion sous la forme d’une application web MERN. Ainsi, elle fonctionne à partir de React sur le front-end, et à partir de Node et d'Express, sur le back-end. De plus, on utilise MongoDB, une base de données NoSQL, pour rendre l’API entièrement dynamique et les données pleinement persistantes.

Code source & Image Docker de l'application

Le code source est accessible via le répertoire GitHub que vous trouverez ici.
Vous pouvez également télécharger l’image de l’application dans le conteneur Docker ici.

Lancement du jeu

Commencer par créer et démarrer les conteneurs avec la commande « docker-compose up ».
Ensuite, ouvrez deux navigateurs et connecter vous à votre port 3000 (‘localhost:3000’).
Vous êtes maintenant prêts à jouer.
